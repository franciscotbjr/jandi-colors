# Stateful Spec for Antigravity

This project uses the **Stateful Spec** methodology.

## Getting Started

1. **Read `.stateful-spec/memory.md`** — your entry point for current project state
2. **Read `.stateful-spec/project-definition.md`** — technology stack, conventions, quality gates
3. **Read `.stateful-spec/methodology/`** — the full methodology (phases, roles, decision framework)

## Operations

Operations are available as workflow files in `.antigravity/workflows/`. Use them to manage the lifecycle of a task:

- `start-session`
- `resume-session`
- `save-session`
- `end-session`
- `create-technical-spec`
- `write-tests`
- `debug-issue`
- `refactor-code`
- `review-changes`
- `write-commit-message`
- `update-documentation`

For more comprehensive rules, refer to `AGENTS.md` at the project root.
